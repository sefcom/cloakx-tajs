Keyboard shortcuts for code viewer

Press To
ALT+G Jump to line.
CTRL+F Start searching
CTRL+G Find next
CTRL+SHIFT+G Find previous
F11 Full screen code viewer (ESC/F11 to exit).
CTRL+D Fold code at cursor (if possible).
CTRL+ARROW-DOWN If jump property selected, jump to line with the next-largest property value.
CTRL+ARROW-UP If jump property selected, jump to the line with the former-largest property value.
CTRL+M Show call hierarchy for the selected line
ALT+M Show related locations for the selected line
